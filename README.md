# practice-logic-diagrams

Logic and algorithm flowchart exercises from various sources.
Includes diagrams, problem statements, and solutions.

## Contents

### Flowcharts
- **algorithm-diagram-exercises.drawio** - Interactive diagrams covering algorithm logic patterns
  - Created with Draw.io for easy visualization and editing
  - Multiple exercises with different complexity levels

### Guides & Solutions
- **guides/** - PDF directory containing:
  - Problem statements (planteamientos)
  - Detailed solutions (soluciones)
  - Step-by-step explanations

---

## Topics Covered

- Control flow logic (if/else, loops)
- Algorithm design patterns
- Flowchart creation and interpretation
- Problem decomposition
- Sequential, conditional, and iterative logic
- Data structure visualization

---

## How to Use

1. **View diagrams**: Open `.drawio` files in Draw.io
2. **Study problems**: Check PDFs in `guides/` for problem statements
3. **Review solutions**: See detailed solutions in `guides/` folder
4. **Practice**: Work through exercises before checking solutions
